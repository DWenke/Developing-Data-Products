## Project Description
Create a web page using R Markdown that features a map created with Leaflet. 

Host your webpage on either GitHub Pages, RPubs or NeoCities.

Your webpage must contain the date that your created the document and it must contain a map created with leaflet. We would love to see you show off your creativity!

### Map
Using mapping data from [Open Street Map](http://www.openstreetmap.org/).
